# Centro-de-Operaço-es-da-Frota-Falcon
# 🚁 Helicopter Fleet Operations System

Sistema de gerenciamento de operações de helicópteros turísticos.

Este projeto foi criado para ajudar na coordenação de voos, controle de combustível, agenda de voos e monitoramento da frota em tempo real.

O sistema também integra com o **Google Calendar** para importar automaticamente os voos agendados.

---

# ✈️ Funcionalidades

- Controle da frota de helicópteros
- Monitoramento de combustível
- Registro de voos
- Agenda de voos
- Radar operacional
- Relatórios de operação
- Histórico de voos
- Integração com Google Calendar
- Cálculo automático de autonomia
- Previsão de abastecimento

# 📊 Painéis do sistema

O sistema possui vários painéis:

### Operações

- Radar da frota
- Status das aeronaves
- Sequência de voos

### Agenda

- Voos agendados
- Confirmação de voo
- Cancelamentos

### Relatórios

- Voos realizados
- Consumo de combustível
- Horas de voo

- previsão de abastecimento
- monitoramento da operação em tempo real
